PETUNJUK PENGGUNAAN ROADMAP
Roadmap ini dirancang agar development SuhastraDev berjalan sistematis dan tidak ada yang terlewat.

Cara Menggunakan Roadmap:
•	Centang (✓) setiap task yang sudah selesai
•	Buat commit Git setelah menyelesaikan setiap FASE
•	Tulis di commit message: "HARI X - FASE Y SELESAI"
•	Jika berhenti di tengah hari, tandai task terakhir dengan catatan

Cara Melanjutkan Development dengan AI:
•	Upload roadmap ini + dokumen perancangan sistem
•	Sebutkan: "Saya sudah menyelesaikan HARI X - FASE Y"
•	Sebutkan task terakhir yang sudah selesai (nomor dan nama task)
•	AI akan melanjutkan dari task berikutnya sesuai roadmap

PENTING: Jangan skip atau ubah urutan task. Setiap task bergantung pada task sebelumnya!

PENDEKATAN ARSITEKTUR:
•	Admin Panel: Controller + Blade tradisional (CRUD form biasa, tanpa Livewire)
•	Frontend Publik: Controller + Blade untuk routing, Livewire Component HANYA untuk:
    - Filter portfolio (kategori)
    - Filter & search produk (kategori, tag, keyword)
    - Form kontak (real-time validation)
•	Validasi: Menggunakan Form Request terpisah untuk setiap form
•	File Upload: Laravel Storage bawaan + Intervention Image (tanpa Spatie Media Library)
